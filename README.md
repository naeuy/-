# Solar Energy Prediction

## 專案結構

- **train_data.csv**: 訓練數據集。
- **requirements.txt**: 套件清單。
- **train.py**: 主程式。
- **predict.py**: 預測用。
- **upload.csv**: 檔案範本。
- **all_gobalred.csv**: 全天空日射量。
- **all_temperature.csv**: 氣溫。

## 環境設定

安裝 **Python 3.12.7** 的版本。

### 步驟 1: 安裝必要套件

執行以下指令安裝專案所需套件：

```bash
pip install -r requirements.txt
```

### 步驟 2: 訓練

```bash
python train.py
```

### 步驟 3: 預測

```bash
python predict.py
```

### 步驟 5 : 上傳檔案

成果 csv，可以上傳!!!
